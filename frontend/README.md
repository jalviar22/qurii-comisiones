# Qurii Comisiones Frontend

React + Vite + TypeScript.

## Desarrollo

```bash
npm install
npm run dev
```

Por defecto apunta al backend en `http://localhost:8000` (ver `.env.development`).

## Build

```bash
npm run build
# output en dist/
```

Variable `VITE_API_URL` define la URL del backend en producción.
